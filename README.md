# terminal-portfolio
An interactive command-line interface portfolio that transforms the traditional web portfolio into a fully functional terminal experience. Built with vanilla JavaScript, CSS, and a monospace aesthetic, it showcases professional experience through intuitive command-based navigation.

Key features include command history (↑↓ arrows), TAB autocomplete, mobile-responsive design with swipe gestures, and a quick-commands panel for touch devices. The entire portfolio runs on vanilla HTML, CSS, and JavaScript with zero dependencies, featuring an authentic green-on-black terminal aesthetic with scanline effects, blinking cursor, and professional ASCII art banner.

Perfect for developers who want to stand out with a portfolio that demonstrates technical creativity while remaining fully functional and accessible.
